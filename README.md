# 🎨 Ronny Site - Mi Página Web

Una página web elegante con galería de imágenes y recursos de fotografía.

## 📋 Características

- ✨ Diseño responsive (se adapta a móviles y escritorio)
- 🖼️ Galería de imágenes con efecto hover
- 🎯 Enlaces a recursos de imágenes gratuitas
- 🎨 Estilos modernos con gradientes
- 📱 Optimizado para todos los dispositivos

## 🚀 Cómo usar

1. Clona este repositorio:
```bash
git clone https://github.com/Rovinsongomez28/ronny-site.git
```

2. Abre el archivo `index.html` en tu navegador

3. Para personalizar:
   - Reemplaza `imagen1.jpg`, `imagen2.jpg`, `imagen3.jpg` con tus propias imágenes
   - Edita el título y contenido según necesites
   - Modifica los colores en la sección `<style>`

## 📁 Estructura del proyecto

```
ronny-site/
├── index.html      # Archivo principal
└── README.md       # Este archivo
```

## 🎨 Colores principales

- Morado principal: `#667eea`
- Morado secundario: `#764ba2`
- Fondo: `#f4f4f4`
- Texto: `#333`

## 🔗 Recursos útiles

- [Unsplash](https://www.unsplash.com) - Imágenes de alta calidad
- [Pixabay](https://pixabay.com) - Banco de imágenes
- [Openverse](https://openverse.org) - Imágenes con licencia abierta

## 📝 Notas

- Asegúrate de que las imágenes estén en la misma carpeta que `index.html`
- Puedes cambiar los tamaños de imagen modificando el CSS
- El sitio es totalmente responsive
